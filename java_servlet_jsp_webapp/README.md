# Web Application Using Servlets and JSP
This project demonstrates a simple web app using Servlets and JSP for user input handling and login validation.

## Part A: User Login Using Servlet and HTML Form
- `login.html` — User input form.
- `LoginServlet.java` — Processes login credentials.
- `welcome.jsp` — Displays personalized welcome message.

## Setup Instructions
1. Place project under `webapps/LoginApp` in Apache Tomcat.
2. Ensure servlet is mapped in `web.xml`.
3. Start Tomcat and open: http://localhost:8080/LoginApp/login.html
